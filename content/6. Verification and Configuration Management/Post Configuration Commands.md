# Post-Configuration Switch Commands 
## Check the active configuration 
	show running-config 
## Check the saved configuration 
	show startup-config 
## See VLANs on the switch 
	show vlan brief 
## Check port status 
	show interfaces status 
## Check trunk links 
	show interfaces trunk 
## Check IP interface status 
	show ip interface brief 
## View the MAC address table 
	show mac address-table 
## Check spanning tree 
	show spanning-tree 
## See nearby Cisco devices 
	show cdp neighbors
## Review system messages
	show logging 
## Save the configuration 
	copy running-config startup-config
