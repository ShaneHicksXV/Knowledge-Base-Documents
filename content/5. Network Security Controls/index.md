# Network Security Controls

## Security is part of basic administration

Security should not be treated like a separate topic that gets added later. On a switch, basic administration already includes several security decisions. If the device is poorly protected, the entire network can become easier to misuse or damage. That is why administrators focus on strong passwords, controlled access, and disabling ports that are not being used. Even a simple switch in a small lab benefits from good security habits because those habits become second nature in larger environments.

## Common protections

A few basic controls make a real difference. One is limiting access to management functions so only trusted users can connect. Another is shutting down unused interfaces so random devices cannot plug in and communicate freely. Clear naming also helps because it reduces confusion when someone audits the configuration. The goal is not to make the switch difficult to use. The goal is to make it dependable, predictable, and safer to manage.

> A secure switch is one that is easy for trusted people to manage and hard for untrusted people to misuse.
