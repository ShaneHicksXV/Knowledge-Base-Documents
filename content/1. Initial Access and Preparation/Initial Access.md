# First-Time Switch Access 

To get into a switch for the first time, the usual move is to connect to the **console port** with a console cable and open a terminal program like PuTTY or HyperTerminal. For many switches, the first login uses a default admin account or a blank password, and then it asks you to create a new password right away. 
## Basic Steps 
1. Connect your laptop or PC to the switch’s console port.  
2. Open a terminal emulator and select the correct serial port. 
3. Use the default settings if needed: 9600 bps, 8 data bits, no parity, 1 stop bit, and no flow control. 
4. Log in with the default admin credentials or press Enter if the password is blank. 
5. Set a new password and start the initial configuration. 
## What to Remember 

The first login is basically just about getting physical access and setting up basic security. After that, you can move into normal configuration like hostname, passwords, and network settings.
